##  Overview
This project is a simple Temperature Monitoring System developed using Python.  
The program simulates an IoT temperature sensor by generating random temperature values at fixed intervals. It compares the generated temperature with user-defined minimum and maximum limits and displays appropriate alert messages. Input validation ensures that the minimum temperature is less than the maximum temperature.

##  Summary
- Accepts minimum and maximum temperature limits from the user  
- Generates random temperature values (0–100) every 2 seconds  
- Compares generated temperature with limits  
- Displays alerts for high or low temperature  
- Shows an error message if minimum limit is greater than maximum limit  

##  Technologies Used
- Python 3  
- random module  
- time module  

##  Concepts Used
- User Input  
- Conditional Statements (if-elif-else)  
- Loops  
- Random Number Generation  
- Time Delay (sleep)  
- Input Validation  

##  Program Logic
1. Accept minimum and maximum temperature values from the user.  
2. Check if minimum temperature is less than maximum temperature.  
3. Generate random temperature values at 2-second intervals.  
4. Compare the generated value with the limits.  
5. Display alert messages for high, low, or normal temperature.
